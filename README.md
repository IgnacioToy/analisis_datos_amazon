# Análisis de datos de productos Amazon

## Objetivo
Evaluar distintas relaciones entre precio, descuentos y valoración de productos
para entender qué factores influyen en la percepción de calidad.

## Proyectos
- Proyecto A: Relación entre descuentos y rating
- Proyecto B: Relación entre precio y rating
- Proyecto C: Relación entre precio, rating y valor percibido

## Datos
Dataset público de productos de Amazon con información de precios, descuentos y valoraciones.

## Herramientas
- Python
- Pandas
- Matplotlib / Seaborn
